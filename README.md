# Sistema de Biblioteca em C

Este projeto é um sistema simples de biblioteca desenvolvido em
linguagem C. O programa permite cadastrar e pesquisar livros durante a
execução, simulando o funcionamento básico de uma biblioteca em memória.

O objetivo principal do código é praticar o uso de structs, vetores, funções
e menus interativos.

## Funcionalidades
- Cadastro de livros com título, autor, código e gênero
- Pesquisa de livros por:
  - Título
  - Autor
  - Gênero
  - Código
- Menu interativo para navegação entre as opções
- Exibição completa das informações dos livros encontrados

## Estrutura do programa
- Uso de `struct` para representar os dados de um livro
- Armazenamento dos livros em um vetor de até 100 posições
- Funções separadas para cadastro e pesquisa
- Menu em loop, permitindo múltiplas operações enquanto o programa estiver ativo

## Objetivo
Praticar conceitos intermediários da linguagem C, como estruturas, manipulação
de strings, funções e organização de código.

## Observação
Este programa foi desenvolvido para fins educacionais e os dados são mantidos
apenas durante a execução do programa.
